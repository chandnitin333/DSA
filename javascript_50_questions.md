# Question 1 - JavaScript Problem (Easy)

## Problem Statement
Write a JavaScript program for interview practice question number 1.

## Solution

```javascript
// Example Solution for Question 1

const nums = [1, 2, 3, 4, 5];

const result = nums.map((num) => {
    return num + 1;
});

console.log(result);
```

```javascript
[2, 3, 4, 5, 6]
```
